# Recipe Manager

A Python-based application for managing and organizing recipes with a simple user interface.

---

## Overview

This project enables users to efficiently add, view, and manage recipes. It combines backend logic with a basic user interface and image handling to deliver a smooth experience.

---

## Features

* Add and view recipes
* Store and manage recipe data
* Display recipe images
* Simple and intuitive interface

---

## Tech Stack

* Python
* HTML, CSS, JavaScript
* Pillow (PIL)

---

## Project Structure

```
recipe_manager/
├── gui/
├── models/
├── utils/
├── web/
├── images/
├── main.py
```

---

## How to Run

```
pip install pillow
python main.py
```

---

## Notes

* File names are case-sensitive (e.g., salad.jpg ≠ Salad.jpg)
* Ensure all required folders are present
